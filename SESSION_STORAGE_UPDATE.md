# تحديث: إظهار نافذة اختيار الدور في كل زيارة

## التغيير المُنفذ:
تم تعديل الموقع بحيث **تظهر نافذة اختيار الدور (طالب / طالب مساعد / دكتور) في كل مرة يفتح المستخدم الموقع**.

## التفاصيل التقنية:

### قبل التعديل:
- كان الموقع يستخدم `localStorage` لحفظ اختيار الدور
- يبقى الاختيار محفوظاً حتى بعد إغلاق المتصفح
- النافذة تظهر مرة واحدة فقط عند أول زيارة

### بعد التعديل:
- الموقع الآن يستخدم `sessionStorage` لحفظ اختيار الدور
- يتم حذف الاختيار تلقائياً عند إغلاق التبويب أو المتصفح
- النافذة تظهر في كل مرة يفتح المستخدم الموقع من جديد

## الملفات المُعدلة:
1. **ui.js** - تم تحديث جميع استدعاءات `localStorage` إلى `sessionStorage`
2. **script.js** - تم تحديث التحقق من دور المستخدم لاستخدام `sessionStorage`

## السلوك المتوقع:
✅ عند فتح الموقع → تظهر نافذة الاختيار  
✅ اختيار الدور → يُحفظ أثناء جلسة التصفح  
✅ التبديل بين الأدوار → يعمل بشكل طبيعي  
✅ إغلاق التبويب/المتصفح → يتم حذف الاختيار  
✅ إعادة فتح الموقع → تظهر النافذة مرة أخرى  

## ملاحظة:
إذا فتح المستخدم الموقع في تبويب جديد (بينما التبويب الأول مفتوح)، ستظهر له نافذة الاختيار في التبويب الجديد لأن كل تبويب له جلسة منفصلة.
